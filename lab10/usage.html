How to Use Web Share API

To use Web Share API is used to share links and text. Using the share() method. To keep the browser from throwing a "TypeError", the content must contain at least one of the following properties: "title, text, url, or files". Text can be shared without a URL or vice versa. Here is an example provided by Web.dev, if (navigator.share) { navigator.share({ title: 'web.dev', text: 'Check out web.dev.', url: 'https://web.dev/', }) .then(() => console.log('Successful share')) .catch((error) => console.log('Error sharing', error)); }
